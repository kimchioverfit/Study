
# tanh 함수 (쌍곡 탄젠트 함수)

## 📌 정의 (수학식)

tanh 함수는 입력값을 -1에서 1 사이로 매핑합니다. Sigmoid와 유사하지만, 중심이 0이라는 장점이 있습니다.

\[
\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}
\]

---

## 📌 그래프

```
    1 |                         ●●●●●●●
      |                    ●●●
      |                ●●●
      |            ●●●
      |        ●●●
    0 |--------●-------------●--------> x
      |       ●●●
      |   ●●●
      | ●●●
   -1 |●●●●●●●
```

---

## 📌 미분

\[
\frac{d}{dx}\tanh(x) = 1 - \tanh^2(x)
\]

---

## 📌 장점

- ✅ 출력 중심이 0이라 학습에 유리
- ✅ Sigmoid보다 gradient 흐름이 더 잘 유지됨

---

## 📌 단점

- ❌ 여전히 Vanishing Gradient 문제 있음 (입력 크면 미분값 0에 수렴)

---

## 📌 활용

- 고전적인 RNN, MLP 등에서 자주 사용
- 현재는 ReLU가 더 보편적이나, 출력 중심을 0으로 유지하고 싶을 때 tanh 사용
