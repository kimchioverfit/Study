
# ✅ 1. 내적 (Dot Product)

### 📌 정의

두 벡터 \( \vec{a}, \vec{b} \in \mathbb{R}^n \)에 대해:

a · b = a₁b₁ + a₂b₂ + ... + aₙbₙ


또는 기하적으로는:


a · b = |a||b|cos(θ)


- \( θ \): 두 벡터 사이의 각도
- \( |a| \), \( |b| \): 각 벡터의 크기

---

### 📐 기하적 의미

- **양의 값**: 서로 0~90도
- **0**: 서로 수직 (직각)
- **음의 값**: 90~180도

---

### 🧮 예제

a = [1, 2, 3]
b = [4, -5, 6]

a · b = 1×4 + 2×(-5) + 3×6 = 4 -10 + 18 = 12


---

### 🔑 성질

- 교환법칙: \( a · b = b · a \)
- 분배법칙: \( a · (b + c) = a · b + a · c \)
- 실수값을 반환
- 내적이 0이면 두 벡터는 **직교(orthogonal)**
