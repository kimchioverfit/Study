# ✅ 2. 외적 (Cross Product)

### 📌 정의

외적은 \( \mathbb{R}^3 \)에서만 정의되며, 두 벡터 \( \vec{a}, \vec{b} \in \mathbb{R}^3 \)에 대해 **새로운 벡터**를 생성합니다.

a × b =
[
a₂b₃ - a₃b₂,
a₃b₁ - a₁b₃,
a₁b₂ - a₂b₁
]


또는 행렬식으로 표현:



a × b = det | i j k |
| a₁ a₂ a₃ |
| b₁ b₂ b₃ |


---

### 📐 기하적 의미

- 결과는 \( a \), \( b \) 두 벡터에 **수직인 벡터**
- 방향은 오른손 법칙 (Right-hand rule)을 따름
- 크기는 두 벡터로 이루는 평행사변형의 넓이:

|a × b| = |a||b|sin(θ)


---

### 🧮 예제

a = [1, 2, 3]
b = [4, 5, 6]

a × b = [
(2×6 - 3×5),
(3×4 - 1×6),
(1×5 - 2×4)
] = [-3, 6, -3]

---

### 🔑 성질

- 반교환법칙: \( a × b = - (b × a) \)
- 분배법칙: \( a × (b + c) = a × b + a × c \)
- 결과는 **벡터**
- 자기 자신과의 외적은 항상 0벡터: \( a × a = 0 \)
